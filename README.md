💼 Job Portal Web Application

A full-stack MERN Job Portal built as a final year project. It allows users to browse jobs, apply, and recruiters to post and manage job listings.

---

🚀 Features

- User Authentication (Login / Signup)
- Browse & Apply for Jobs
- Post & Manage Jobs (Recruiter)
- View Applications

---

🛠️ Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Others: JWT, Axios

---

📂 Project Structure

job-portal/
├── frontend/
├── backend/
└── README.md

---

⚙️ Setup & Run

git clone https://github.com/Priyanshi-678/jobportal.git
cd job-portal

Install dependencies

cd backend && npm install
cd ../frontend && npm install

Run project

# backend
cd backend && npm run dev

# frontend
cd frontend && npm run dev

---

🔐 Environment Variables

Create a ".env" file in backend:

MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
